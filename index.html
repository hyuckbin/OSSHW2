Hyuckbin의 과제 포트폴리오

[HTML, CSS, JavaScript]를 이용한 Form 페이지 제작 프로젝트]

👋 소개

안녕하세요! 컴퓨터공학 심화를 공부하고 있는 고혁빈입니다. 이번 프로젝트를 통해 웹 페이지의 기본이 되는 Form을 만들고, CSS로 디자인하며, JavaScript로 동적인 기능을 추가하는 연습을 했습니다.

🚀 과제 설명

Lab 1: Form Tag 연습 (HTML)

웹 페이지의 핵심인 폼(Form)을 구성하는 다양한 HTML 태그를 연습했습니다. <input>, <select>, <legend>, <fieldset> 등 시맨틱 태그를 활용하여 주문/결제 페이지의 기본 구조를 만들었습니다.

페이지 보기

Lab 2: Form CSS 적용

Lab 1에서 만든 HTML 구조에 CSS를 적용하여 디자인을 개선했습니다. Flexbox를 이용한 중앙 정렬, 사용자 인터랙션을 고려한 :hover, :focus 효과, 그리고 전반적인 색상과 레이아웃을 다듬어 사용성을 높였습니다.

페이지 보기

Lab 3: Form Javascript 적용

HTML과 CSS로 만든 정적인 페이지에 JavaScript를 더해 동적인 기능을 구현했습니다. '결제하기' 버튼 클릭 시 각 입력 항목의 값이 유효한지 검사하는 유효성 검사(Validation Check) 기능을 추가하여 사용자의 실수를 방지하도록 만들었습니다.

페이지 보기

Lab 5: 종합 연습

주어진 요구사항에 맞춰 기존 Form 페이지를 수정하는 종합 연습을 진행했습니다. 색상 테마 변경, 라디오 버튼 커스터마이징, 필수 입력 조건 추가 등 HTML, CSS의 다양한 속성을 응용하여 실무적인 문제 해결 능력을 길렀습니다.
